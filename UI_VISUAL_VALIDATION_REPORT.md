## 3. Horizontal scroll (320px)

- **Body/html:** No `overflow-x: hidden` on `body` or root layout. If any component renders wider than the viewport, page-level horizontal scroll can occur.
- **PartnershipSection:** Uses `overflow-x-auto` inside a constrained area (logo strip). Intentional; not page-level overflow.
- **programs/utr-match-play:** Uses `overflow-x-auto` on a table wrapper. Contained.

**Recommendation:** Add `overflow-x-hidden` to `body` (e.g. in `globals.css` or layout) as a safeguard at 320px and other breakpoints. No horizontal scroll was observed in the 320px snapshot; this is a hardening step.
