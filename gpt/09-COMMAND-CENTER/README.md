# 09 — Command Center

> The single place to start, run, and operate the LBTA Custom GPT system. Everything else in this repo is reference; this folder is the cockpit.

---

## What lives here

| File | Purpose | When to use |
|---|---|---|
| `master-setup-checklist.md` | Step-by-step build of all 6 GPTs from zero | First-time setup. Print it, work through it. |
| `daily-cockpit.md` | The 60-second "is everything healthy?" check | Andrew opens this each morning during dogfood week, weekly thereafter. |
| `weekly-cockpit.md` | The Friday Compound Review opening shot | Every Friday before the team ritual. |
| `quick-reference.md` | Phone, email, emergency commands, file locations | Pin to desk. Speed up moments where you're hunting for a fact. |

---

## How to use this folder

**If you've never built a GPT here:** open `master-setup-checklist.md`. Work top to bottom. Don't skip Phase 1 to "save time" — it compounds.

**If you're operating the system day-to-day:** open `daily-cockpit.md` (each morning) and `weekly-cockpit.md` (each Friday). They are designed to be 60 seconds and 5 minutes respectively.

**If you're hunting a number, a runbook, or a contact:** `quick-reference.md`.

---

## Source-of-truth

Everything in `09-COMMAND-CENTER/` references files in `01-` through `08-`. If a number, link, or step here looks stale, check the upstream file first — that's the source of truth.

If the upstream and the cockpit disagree, **upstream wins**. Update the cockpit to match.
