# LBTA Custom GPTs — Communications

> Internal and external messaging about the GPT system. Calm, specific, no hype.

---

## Files

| File | Audience | Use when |
|---|---|---|
| `team-launch-announcement.md` | Allison, Saska | Day 1 — they're getting their GPT |
| `coach-1on1-talking-points.md` | Andrew → 1 coach | Adoption stalls, voice drifts, or to compound a win |
| `parent-faq.md` | Members / parents | They ask "are you using AI?" |

## Voice rules for these comms

- Same brand voice: calm, specific, no marketing speak.
- No "AI" hype. The GPTs are tools. Tools get used or not.
- Coaches are not "users" — they're coaches who use a tool.
- Parents are not "customers" — they're members.

## What this folder is NOT

- Not an HR playbook. (See coach contracts, separately.)
- Not a privacy / legal document. (See `07-LEGAL-PRIVACY/`.)
- Not training material. (Coaches don't get trained on AI — they get a tool that drafts in their voice.)

## When to add a file here

If a new audience needs a clear message:
- Vendors / contractors asking how AI is integrated → add `vendor-faq.md`.
- Members asking the same question → use `parent-faq.md` (works for both).
- Press / media → escalate to Andrew, then add a draft template here.
