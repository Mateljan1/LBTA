## Design Principles Applied

### 1. Asymmetry Over Symmetry
- 7 + 5 column layouts (not 6 + 6)
- 3 + 2 column layouts (not even split)
- Full-bleed images (not contained)

### 2. Typography as Art
- Font sizes: 7rem, 9rem headlines
- Font weights: 200, 300 (ultra-light)
- Negative letter spacing (-0.03em, -0.04em)
- Serif for drama, sans for clarity

### 3. Photography Drives Narrative
- Full-bleed images
- Text overlaid on photos
- Images aren't decorative‚Äîthey ARE the section
- Minimal captions, maximum impact

### 4. Rhythm Variation
- Full-height hero (100vh)
- Full-bleed ATP story (min-h-screen)
- Tall philosophy section (py-40)
- Magazine programs (70vh + 60vh)
- Portrait feature (2/3 aspect)
- Centered video moment
- Dramatic CTA (70vh)

### 5. Minimal UI Elements
- No borders (except subtle dividers)
- No shadows (except video)
- No cards
- No boxes
- Just content and white space

### 6. Confident Simplicity
- One button in hero (not two)
- Understated links (not loud CTAs)
- Remove 40% of previous content
- What remains: perfect

## Technical Implementation

### Key CSS Patterns:

```css
/* Ultra-light typography */
fontWeight: 200, 300
letterSpacing: '-0.03em', '-0.04em'

/* Asymmetric grids */
lg:grid-cols-12 (7 + 5 split)
lg:grid-cols-5 (3 + 2 split)

/* Full-bleed images */
-mx-6 md:-mx-16 lg:mx-0

/* Editorial spacing */
py-32 md:py-40 (not uniform section-spacing)

/* Minimal interactions */
hover:scale-105 duration-[1500ms]
opacity-40 hover:opacity-80

/* No shadows (except video) */
Only one shadow in entire page

/* No borders (minimal dividers only) */
border-t border-gray-200 (subtle)
```

### Removed Classes:
- `card-lbta` (no more cards!)
- `border-l-2` on programs (was corporate)
- Heavy shadows
- Background boxes
- Grid patterns everywhere

### Added Techniques:
- Full-viewport sections (`h-screen`, `min-h-screen`)
- Asymmetric column grids (`lg:grid-cols-12`)
- Overlaid text on images
- Floating logos (no containers)
- Variable section heights

- Junior: 70vh hero image

- Hero: 100vh
- ATP: min-h-screen

### 4. Typography as Design Element
Oversized numbers and headlines:
- 9rem ATP ranking

- Ultra-light weights (200, 300)

### 5. Minimal UI

- All shadows (except video)

- All background boxes
- All icons

### 6. Photography-Driven
Images aren't decoration:
- They ARE the section
- Text integrated into images
- Full-bleed, edge-to-edge

## Mobile Optimization

All layouts adapt beautifully:

### Full-Bleed Sections:
- Desktop: Full-width edge-to-edge
- Mobile: `-mx-6` pulls to edges, maintains full-bleed feel

### Asymmetric Grids:
- Desktop: 7+5, 3+2 columns
- Mobile: Stack vertically, maintain hierarchy

### Typography:
- Desktop: 7rem, 9rem headlines
- Mobile: clamp() scales down proportionally
- Always readable, always elegant
