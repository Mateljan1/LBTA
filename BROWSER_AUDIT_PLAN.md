# LBTA Browser Automation Audit Plan

## Pages to Test
1. Homepage (/)
2. About (/about)
3. Programs (/programs)  
4. Schedules (/schedules)
5. Coaches (/coaches)
6. Contact (/contact)
7. Fitness (/fitness)
8. Camps (/camps)

## For Each Page
1. Navigate to URL
2. Take desktop screenshot (1440px)
3. Take tablet screenshot (1024px)
4. Take mobile screenshot (430px)
5. Verify hero overlay visible
6. Check text contrast
7. Test interactive elements
8. Verify sticky CTAs on mobile

## Key Checks
- Hero gradients visible
- Text readable on all viewports
- Buttons accessible and styled correctly
- Grid layouts responsive
- Forms functional
- Animations smooth
